
1.sudo apt-get install libassimp-dev // подключение бибилиотеки assimp

2.sudo apt-get install libglfw3 // подключение библиотеки glfw

3.Также вам надо компилировать файл следующим образом
g++ main.cpp glad.o -o main -lglfw -ldl -lassimp
4.И запускать ./main
4.Инструкция по управлению симуляцией:
   Клавиша W движение по направлению курсора;
   Клавиша S движение в направлении противаположном курсору;
   Клавиша A движение влево;
   Клавиша D движение вправо;
   Клавиша UP движение перпендикулярно направлению на курсор, "вверх";
   Клавиша DOWN движение перпендикулярно направлению на курсор, "вниз";
   Клавиша RIGHT ускорение движения объектов;
   Клавиша LEFT замедление движения объектов;
   Клавиша Q возвращение в первоначальное положение внуртри Солнца(0, 0, 0);
   Курсор наклон камеры;
